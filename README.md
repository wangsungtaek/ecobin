# ecobin (1차 팀프로젝트)
#### 개요
> 친환경 제품을 판매하는 쇼핑몰 사이트 제작
#### 기간
> 21.01.11 ~ 21.01.28
#### 담당영역
> 마이페이지
#### 목표
1. 웹사이트의 기본 구조가 되는 HTML을 사용하여 정적인 페이지를 제작
2. CSS를 통해 원하는 디자인 및 레이아웃 가능
3. 차후 동적인 페이지를 만들기 위해 필요로 하는 데이터베이스 설계
